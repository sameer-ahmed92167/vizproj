üìä Project Description

This project is an interactive web dashboard for analyzing motor vehicle collisions in New York City. The dashboard provides temporal visualizations of accident patterns, allowing users to explore:

  • Collision trends over time (daily, hourly patterns)
  • Geographic hotspots of injuries and fatalities
  • Contributing factors analysis (Vehicle 1 & 2 factors)
  • Interactive filtering by injury severity, time of day, and location

The dashboard transforms raw NYC collision data into actionable insights for urban planning, traffic management, and public safety initiatives.

üìÅ Dataset Description

NYC Motor Vehicle Collisions - Crashes

Source: NYC Open Data Portal

Time Period: 2012-2024

Size: 800000+ records with 17+ features

Key Features:

  • Temporal Data: Crash dates and times for time-series analysis
  • Geographic Data: Latitude/Longitude coordinates for mapping
  • Injury Metrics: Persons injured, pedestrians/cyclists/motorists affected
  • Contributing Factors: Primary and secondary contributing factors
  • Location Details: Borough, street names, ZIP codes

The dataset is updated regularly by the NYPD and provides comprehensive coverage of reported collisions across all five boroughs.
